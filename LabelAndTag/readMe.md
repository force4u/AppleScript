### 【Ｂ】ラベル  
【１】ラベル無し  
【２】ラベル単色  
【３】ラベル複数色＝ラベル単色＋色ラベルタグ×複数  
  
### 【Ｃ】タグ  
【１】タグ無し  
【２】色ラベル無しタグ  
【３】色ラベル有りタグ  
【４】複数色ラベルタグ（ラベルの３ラベル複数色と同じ）
